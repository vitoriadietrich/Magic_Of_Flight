# ✈️ Magic of Flight

**Magic of Flight** é uma **landing page imersiva sobre viagens e aviação**, inspirada no design do Figma, com **vídeo animado de avião** e elementos visuais que trazem sensação de movimento pelo céu. 🌤️  

---

## ✨ Destaques do Projeto

- 🛫 **Vídeo do Avião:** Animação integrada do Figma que mostra o avião em movimento.  
- 🎨 **Design Clean e Elegante:** Fundo claro com detalhes sutis, tipografia moderna e layout espaçoso.  
- 🌍 **Tema de Viagem:** Conteúdo pensado para transmitir liberdade e aventura.  
- 📱 **Responsivo:** Layout adaptável para desktop, tablet e mobile.  
- 🎯 **Call-to-Action:** Botões claros e chamativos, convidando o usuário a explorar mais.  

---

## 💻 Tecnologias Utilizadas

| Tecnologia | Uso |
|-----------|-----|
| HTML5     | Estrutura da página |
| CSS3      | Layout, cores, animações e efeitos |
| JavaScript| Interações e comportamento dinâmico |
| Vídeo Figma | Animação do avião para destaque visual |
